# 🛒 Mini Data Warehouse Project - E-Commerce Data Analysis

## 📖 About the Project
This is my first personal project on building a small-scale Data Warehouse. The project simulates the process of designing a relational database from scratch, including creating table structures, loading raw data, and writing SQL queries to extract business insights from e-commerce transactions.

## 🛠️ Tech Stack
* **Database Management System:** SQLite
* **Tool:** DB Browser for SQLite
* **Language:** SQL (DDL for table creation, DML for data insertion, DQL for querying)

## 🗄️ Database Schema
The system follows a standard relational data model, consisting of 3 main tables tightly linked via Primary Keys and Foreign Keys:
1. **Customers:** Stores customer identification information.
2. **Products:** Product catalog and pricing.
3. **Orders:** Records actual transaction history.
